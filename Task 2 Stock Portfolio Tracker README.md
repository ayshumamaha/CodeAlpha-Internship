# Stock Portfolio Tracker

## Overview

The Stock Portfolio Tracker is a Python-based financial management application designed to monitor stock investments and calculate portfolio value. Users can track purchased stocks, quantities, and overall investment performance.

The project demonstrates data management, calculations, and financial application development using Python.

---

## Features

* Add stock information
* Track investment holdings
* Calculate portfolio value
* Display investment summary
* Manage multiple stocks
* Organized portfolio tracking

---
