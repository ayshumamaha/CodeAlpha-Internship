# Hangman Game

## Overview

The Hangman Game is a Python-based word guessing game where players attempt to identify a hidden word by guessing individual letters. The player is given a limited number of incorrect attempts before the game ends.

This project demonstrates the implementation of game logic, string manipulation, loops, conditional statements, and user interaction using Python.

---

## Features

* Random word selection
* Letter-by-letter guessing
* Limited attempts system
* Win/Loss detection
* Interactive gameplay
* User-friendly interface

## Workflow

1. Select a random word
2. Display hidden word format
3. Accept user guesses
4. Validate guessed letters
5. Update displayed word
6. Reduce attempts for wrong guesses
7. Determine win or loss
